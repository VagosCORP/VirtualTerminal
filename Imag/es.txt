Use Bluetooth/WiFi para controlar cualquier proyecto de pasatiempo o profesional

Virtual Terminal permite iniciar una comunicación basada en sockets a travez de protocolos inalámbricos, tanto WiFi como Bluetooth, presentes en la mayoría de dispositivos. El otro extremo de la comunicación puede ser un microcontrolador o Arduino con el hardware inalámbrico apropiado (módulo HC-06 es un ejemplo para bluetooth), otra aplicación o software de computadora que provea un servidor al cual conectarse. Esta aplicación fue desarrollada pensando en proyectos integrados tanto de pasatiempo como a nivel profesional, buscando una interfaz gráfica completa y flexible que permita al usuario controlar y depurar su trabajo lo mas fácil y comodamente posible.
Virtual Terminal incluye las siguientes características de configuración:
• Enviar Texto como Bytes
• Enviar Entero como Byte
• Enviar Entero como Byte(escritura binaria)
• Enviar Entero como Byte(escritura hexadecimal)
• Nuevas opciones avanzadas de envío:
  - Envíe enteros de 16bits como 2 Bytes
  - Envíe enteros de 32bits con signo como 4 Bytes
  - Envíe números flotantes con signo como 4 Bytes (32bits)
• Desplazamiento automático de la sección de recepción (RX)
• Recibir Bytes como Caracteres ASCII
• Recibir Bytes como Enteros
• Recibir Bytes como Caracteres y Enteros al mismo tiempo con la nueva opción 'Dual' 
• Recibir Información constantemente actualizada en tiempo real mediante la nueva opción 'Actualización'
  - Requiere caracter de finalización "\n" en cada cadena de datos
  - La nueva información sobrescribirá la antigua, permitiendo altas velocidades de actualización
  - La entrada '#Act' le permitirá acumular y ver mas de una cadena de información a la vez
• Nuevas opciones avanzadas de recepción: (requiere carácter \r de inicio y \n de finalización)
  - Reciba enteros de 16bits como 2 Bytes
  - Reciba enteros de 32bits con signo como 4 Bytes
  - Reciba números flotantes con signo como 4 Bytes (32bits)
• Modo Comando Mejorado:
  - Habilita un grupo de botones configurables
  - Pulse prolongadamente para guardar un comando desde la casilla de envío (si la casilla esta en blanco se borrará el comando existente)
  - Pulsación simple para enviar el comando guardado
  - Los botones de comando mostrarán tanto el valor como el tipo de dato almacenado (Texto, Binario, Flotante, etc.)
• Selector de apariencia, habilitando un tema 'Dark' para proteger su vista

Notas:
- Bluetooth: Vincule el dispositivo con el que desea trabajar (en ajustes de Buetooth) antes de usarlo.
- WiFi: Debe conocer la dirección IP y el puerto del dispositivo para poder establecer la conexión.
